Data Structure & Algorithms

You hate it in the beginning (in first second year of the college), somehow passing the subject and praying that you'll not have to face it again.

Data structure keeps coming back to you whatever courses you may take, be it algorithms be it artificial intelligence. I was playing hide n seek with data structure during my whole third year.

Comes fourth year, its placement time. You got rejected in a few interviews because you didn't know data structures, you sit down muster some courage and wooohhhhaaaaa!!! you love it

Books

Fundamentals of Data Structures

by Ellis Horowitz

Description

This new edition provides a comprehensive and technically rigorous introduction to data structures such as arrays, stacks, queues, linked lists, trees and graphs and techniques such as sorting hashing that form the basis of all software. In addition, this text presents advanced or specialized data structures such as priority queues, efficient binary search trees, multiway search trees and digital search structures. Description

Thus, this book will be an excellent self-teach and test preparation material for beginners.
