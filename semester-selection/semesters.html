1st Year

Things you should do in 1st Year

  Learn programming

  Practice public speaking - join competitions like debate, extempore speech etc.

  Enhance your writing skills - write wikipedia

2nd Year

Things you should do in 2nd Year

  A great command over English, written and verbal (English)

  Great problem solving skills and logical reasoning (Aptitude & Reasoning)

  Pick up a single language from Java / Php / Ruby / Python / C# and master it. Just a single one, dont run for all. Mastering a single language make you better eligible for a job.

  Subjects like C, DBMS, Data Structure, Algorithms, Computer Networks and Operating System.

03

Semester Three

  Start Competitive Programming

  Regularly participate in coding contests - Code Chef, Hacker Earth etc.

  Start Creating your resume
